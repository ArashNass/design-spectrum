# Seismic Design Spectrum Generator

**Live tool:** https://arashnassirpour.com/design-spectrum/

Generate and compare elastic and reduced seismic response spectra across international structural-design standards.

## Supported standards

- Eurocode 8 / EN 1998-1:2004.
- ASCE 7-22.
- NZS 1170.5.
- AS 1170.4.
- National Building Code of Canada.
- Türkiye Building Earthquake Code 2018.
- IS 1893.
- GB 50011.
- NTC-Sismo.
- Iran Standard 2800.

The available parameters, equations, labels, and spectrum shape update with the selected standard. Results can be visualised, compared, and exported as CSV.

## Run locally

The application is a standalone client-side `index.html` with no build step. Open it directly in a modern browser or serve the repository with any static web server.

## Deployment

GitHub Pages publishes `main` at the live URL above.

## Engineering responsibility

This tool supports exploration and independent checking. Users remain responsible for confirming code editions, national annexes, jurisdictional amendments, hazard parameters, site classification, importance factors, damping, behaviour/reduction factors, and project-specific requirements. It is not a substitute for a qualified structural engineer or the official standards.
