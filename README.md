# Seismic Design Spectrum Generator

Live site: https://arashnassirpour.com/design-spectrum/

Generate and compare elastic seismic response spectra across Eurocode 8, ASCE 7-22, NZS 1170.5, AS 1170.4, NBC, TBDY 2018, IS 1893, GB 50011, NTC-Sismo, and Iran Standard 2800.

The application is a standalone, client-side `index.html` hosted with GitHub Pages. It is intended for engineering exploration and independent verification; users remain responsible for confirming inputs, code editions, national annexes, and project-specific requirements.

## Run locally

Open `index.html` in a browser, or serve this directory with any static web server.
